01
Cloning Engine
Single-file Playwright-based web cloner. Captures CSS, images, fonts, videos, 3D models, WebGL shaders, and animations. Detects framework (Next.js, Webflow, React) and applies specialized handling.
Playwright Single File -- lines
02
Test Suite
Parallel pipeline: clone→score→analyze per site. 9 strict metrics with pixel-level comparison (pixelmatch). Generates per-site analysis reports and cross-site synthesis.
9 Metrics Pixel Diff Parallel
03
Safety Systems
PID lock prevents overlapping cycles. 90-min timeout kills stuck processes. Per-site regression gate rejects if any site drops >5 points. Disk cleanup keeps last 5 results.
PID Lock Timeout Regression Gate
04
Learning System
Knowledge base persists across cycles. Records techniques that worked (with impact scores) and approaches that failed (with reasons). Claude reads this before every cycle.
